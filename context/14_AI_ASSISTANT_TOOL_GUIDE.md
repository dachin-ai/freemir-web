# Freemir AI Assistant Tool Guide

This file is used as a practical knowledge source for the in-app AI assistant.

## Main Purpose

The assistant helps users:
- choose the correct tool for their task,
- understand how each tool works,
- and follow the right steps in the app.

## Tool Routing (Quick)

- Price checking, bundle margin, target stock:
  - use `Price Checker`
- Daily order estimation by warehouse/platform:
  - use `Order Planner`
- Audit for order loss and profitability:
  - use `Order Review`
- Shopee affiliate data checks and reports:
  - use `Affiliate Performance`
- TikTok pre-sales volume planning:
  - use `Pre-Sales Checker`
- TikTok affiliate analytics:
  - use `Affiliate Analyzer`
- TikTok ads data analysis:
  - use `Ads Analyzer`
- Product performance upload/compute/compare:
  - use `Product Performance`
- Livestream showcase and related sync:
  - use `Livestream Display`

## Price Checker Guidance

When users ask "cek harga", "aman tidak", or mention SKU:
- explain they should open `Price Checker`,
- mention available modes (`Batch` and `Direct Input`),
- and for direct checks, explain SKU input separators:
  - plus (`+`)
  - comma (`,`)
  - whitespace

If users ask about campaign tiers:
- explain `Warning` is the base / top safe reference,
- campaign outputs (`Daily-Discount`, `Daily-Livestream`, etc.) are comparison targets,
- `Invalid` means SKU not found in loaded price DB.

## Response Style

- Keep answers clear and action-oriented.
- Use short numbered steps when explaining app flow.
- If data is missing, state limitations and next action explicitly.
